# MathJax Example App

Example Expo application demonstrating the usage of `react-native-mathjax-html-to-svg`.

## Get Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Start the app
   ```bash
   npx expo start
   ```

3. Open the app in:
   - iOS Simulator (press `i`)
   - Android Emulator (press `a`)
   - Expo Go app (scan QR code)

## What's Included

This example demonstrates various mathematical formulas:
- Einstein's mass-energy equivalence: E = mc²
- Pythagorean theorem: a² + b² = c²
- Quadratic formula
- Euler's identity
- Gaussian integral
